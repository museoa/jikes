Jikes on Mac

Jikes has been successfully compiled and used on MacOS X 10.2 (Jaguar).

Development tools

You'll need Apple's Developer Tools to compile Jikes. Although they come with Mac OS X 10.2, they're not installed by default. You can install them from the gray Developer Tools CD-ROM that accompanied the other Mac OS X CD-ROMS, or you can download a disk image free from Apple's Developer website.

The Developer Tools installation includes the GUI tools such as ProjectBuilder and many of the underlying GNU tools you know and love from other Unixes, including all the ones you'll need to build Jikes.

The command-line tools are installed in /usr/bin/ which will already be on your PATH.

Building Jikes

Once you've installed the Apple Developer Tools, you can simply follow the generic compiling instructions in the Jikes Developer FAQ.
