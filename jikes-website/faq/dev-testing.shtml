Testing Jikes

 

Jikes makes use of a regression test framework called Jacks. This suite of tests helps to ensure that already fixed bugs stay fixed and that new code is tested as it is added. Check out the Jacks project page for more info about how to setup and run the Jacks tests. Most users will not have any reason to use Jacks, but developers contributing code to jikes or working on new ports should be familiar with it.
