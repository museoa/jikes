The Jikes Project keeps all our source files, build environment, documentation, and web pages in CVS. The information below should be sufficient to get those familiar with CVS up and running, additional information about CVS can be obtained here.

CVSROOT
:pserver:anonymous@cvs.sourceforge.net:/cvsroot/jikes
CVSROOT for developers
:ext:USER@cvs.sourceforge.net:/cvsroot/jikes
You'll also need to set CVS_RSH=ssh. more info from sf.net

jikes
The main compiler source code
